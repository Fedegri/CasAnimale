COMANDI UTILI
Per creare branch locali collegate alle branch remote (tutte) <3
	git branch -r | grep -v '\->' | sed "s,\x1B\[[0-9;]*[a-zA-Z],,g" | while read remote; do git branch --track "${remote#origin/}" "$remote"; done

Gestione utenti
FrontOffice: Utente Non Registrato -> Utente Registrato -> Utente vip (sblocca alcuni prodotti e servizi)
BackOffice: Utente Aziendale (vede il pannello di amministrazione delle propria azienda) -> Administrator (vede tutto)

Utente vip può:
  - comprare animali e cuccioli
  - prenotazione a servizi costosi (wedding planner, pensioni)
  - servizi online
